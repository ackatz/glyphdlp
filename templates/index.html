Glyph DLP is a data loss prevention API that can be used to scan or redact sensitive data from documents.

Try our demo Get started for free

How does it work?

Step 1 | Send an API Call

Send Glyph a Base64-encoded document via our secure REST API (we do NOT track or store any file content or metadata)

Step 2 | Scan or Redact

Glyph parses your document, with your choice of scanning or redacting sensitive information

Step 3 | Secured

Receive the scan results or redacted document, now safer and compliant with your data handling protocols

Supported File Types

  • Text file (.txt)
  • Word document (.docx)
  • Excel spreadsheet and CSV (.xlsx, .csv)
  • PDF (.pdf)
  • JSON (.json)
  • See our docs for more supported file types

Supported Detections

  • Credit card numbers
  • Social security numbers
  • Bank account/routing numbers
  • Date of birth
  • Email addresses
  • See our docs for more supported detections
